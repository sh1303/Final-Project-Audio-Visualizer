Final Project Check-In — Music Visualizer (OpenGL + C++)

Project Description

For our final project, Nirshal Chandra Sekar and Samra Huseynova are going to implement a real-time 3D music visualizer using only OpenGL and C++. The goal is to create a dynamic visual scene that responds to audio in real time through movements of different shapes, color changes, lighting variations, and if time allows, we plan the inclusion of particle effects too.

The system will read an audio file, extract raw samples, compute the frequency spectrum using an FFT implementation, and map different frequency bands to visual elements. The main visual components will include a set of 2D/3D bars (or other shapes) whose heights and colors correspond to frequency magnitudes, along with a reactive particle system where emission rate, velocity, and color shift are tied to beat intensity. Adjusting camera pose and adding basic lighting will enhance the overall look, making the visualization feel alive and energetic.

This project includes substantial graphics programming (real-time geometry updates, shaders, transformations, lighting, buffer management) and is going to produce a final output that will be different from our previous assignments.

Progress So Far

We started by having a look at some existing cool works for audio visualization. Based on the insights obtained, we have sketched the steps to complete our project. As the initial step, we started with the analysis of audio files. As the next step, we plan to move on to the implementation of FFT.

Expected Result

The final visualizer will display a set of animated bars arranged either in a line or circular pattern. These bars will scale up and down based on the magnitude of the corresponding audio frequencies. Particles will burst outward during strong beats, gradually fading as they move. The scene will use dynamic lighting, color shifts, and a moving camera to create an immersive experience.
